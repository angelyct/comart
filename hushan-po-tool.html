主旨
此工具會呼叫 Claude API 解析PDF採購單，並透過 Google Drive（HU#／COMART#對照表）與 Microsoft 365（Outlook草稿）等連接器完成自動化流程。這些功能需要在 Claude.ai 對話中以 Artifact 形式開啟才能運作； 若另存為檔案後在本機直接開啟，自動解析與寄信功能將無法使用，但仍可手動編輯品項並產生 Excel／Email內容、下載檔案。
若「嘗試透過 Microsoft 365 建立草稿」目前沒有可用工具，請使用下方的「複製郵件內容」或「在Outlook網頁版開啟」作為備用方式，並記得手動附上 PDF 與 Excel 兩個附件。
